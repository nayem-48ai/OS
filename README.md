অবশ্যই! আপনার GitHub Actions 워크플로우-এর জন্য একটি বিস্তারিত README.md ফাইল নিচে তৈরি করে দেওয়া হলো। এটি Markdown ফরম্যাটে লেখা হয়েছে, তাই আপনি সরাসরি আপনার রিপোজিটরিতে এটি ব্যবহার করতে পারবেন।

GitHub Actions এর মাধ্যমে Tailscale Funnel সহ SSH অ্যাক্সেস

এই GitHub Actions 워크플로우 ব্যবহার করে আপনি একটি অস্থায়ী (temporary) Ubuntu ভার্চুয়াল মেশিনে SSH অ্যাক্সেস তৈরি করতে পারবেন, যা Tailscale এবং Tailscale Funnel দ্বারা সুরক্ষিত এবং ইন্টারনেটে উন্মুক্ত থাকবে। এর মাধ্যমে আপনি যেকোনো জায়গা থেকে আপনার GitHub Actions রানারে চলমান একটি সম্পূর্ণ লিনাক্স পরিবেশে কাজ করতে পারবেন।

প্রধান বৈশিষ্ট্য (Features)

ম্যানুয়াল ট্রিগার: workflow_dispatch ব্যবহার করে যেকোনো সময় 워크플로우 চালানো যায়।

স্বয়ংক্রিয় সেটআপ: SSH সার্ভার এবং Tailscale ক্লায়েন্ট স্বয়ংক্রিয়ভাবে ইনস্টল এবং কনফিগার হয়।

Tailscale Funnel ইন্টিগ্রেশন: 워크플로우 শুরু হওয়ার সাথে সাথেই স্বয়ংক্রিয়ভাবে Tailscale Funnel চালু হয়ে যায়, যার ফলে SSH সার্ভারটি পাবলিক ইন্টারনেট থেকে অ্যাক্সেস করা যায়।

কাস্টমাইজযোগ্য: SSH ইউজারনেম, পাসওয়ার্ড, রানটাইম, এবং ইনস্ট্যান্স সংখ্যা নিজের মতো করে সেট করা যায়।

স্বয়ংক্রিয় পরিষ্করণ: পুরনো Tailscale ডিভাইসগুলি স্বয়ংক্রিয়ভাবে মুছে ফেলার অপশন রয়েছে।

একাধিক ইনস্ট্যান্স: একই সাথে একাধিক SSH সেশন (সর্বোচ্চ ১০টি) চালানো সম্ভব।

پیش نیاز (Prerequisites)

এই 워크플로우 ব্যবহার করার জন্য আপনার নিচের জিনিসগুলো প্রয়োজন হবে:

GitHub Account: GitHub Actions চালানোর জন্য।

Tailscale Account: একটি ফ্রি Tailscale একাউন্ট।

Tailscale API Key:

আপনার Tailscale Admin Console থেকে একটি API কী তৈরি করুন।

এই কী-টির ডিভাইস অ্যাডমিন হিসেবে কাজ করার অনুমতি (read এবং write অ্যাক্সেস) থাকতে হবে, যাতে এটি পুরনো ডিভাইস মুছে ফেলতে পারে।

Tailscale Auth Key:

আপনার Tailscale Admin Console এর Keys সেকশন থেকে একটি Auth Key তৈরি করুন।

এটি একটি Reusable বা Ephemeral কী হতে পারে। দীর্ঘ সময় ব্যবহারের জন্য Reusable কী ব্যবহার করা ভালো।

কিভাবে ব্যবহার করবেন (How to Use)

রিপোজিটরি সেটআপ:

এই রিপোজিটরিটি ফর্ক করুন অথবা .github/workflows/ ফোল্ডারের মধ্যে থাকা .yml ফাইলটি আপনার নিজের রিপোজিটরিতে যোগ করুন।

GitHub Secrets তৈরি করুন:

আপনার রিপোজিটরির Settings > Secrets and variables > Actions এ যান এবং নিচের সিক্রেটগুলো তৈরি করুন। এটি আপনার সংবেদনশীল তথ্য সুরক্ষিত রাখবে।

TS_API_KEY: আপনার Tailscale API কী।

TS_AUTHKEY: আপনার Tailscale Auth Key।

SSH_PASSWORD: আপনার পছন্দের একটি শক্তিশালী SSH পাসওয়ার্ড।

워크플로우 চালান:

আপনার রিপোজিটরির Actions ট্যাবে যান।

বাম পাশের সাইডবার থেকে "SSH + Tailscale with Funnel" 워크플로우টি নির্বাচন করুন।

"Run workflow" ড্রপডাউন বাটনে ক্লিক করুন।

প্রয়োজনীয় ইনপুটগুলো পূরণ করুন (নিচে বিস্তারিত আলোচনা করা হয়েছে) এবং "Run workflow" বাটনে ক্লিক করুন।

কিভাবে সংযোগ করবেন (How to Connect)

워크플로우 সফলভাবে রান হওয়া শুরু করলে:

Actions ট্যাবে চলমান 워크플로우-এর ssh_job ধাপটিতে যান।

লগ আউটপুটটি விரிস্তভাবে দেখুন।

"Setup SSH Server and User" ধাপের আউটপুটে আপনি সংযোগ করার জন্য প্রয়োজনীয় তথ্য পাবেন।
